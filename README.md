Importante!!

Para acceder al dashboard por medio de la instancia de ec2 es necesaria la ip publica dada por aws. Dicha IP publica cambia cada vez que se pausa la instancia. 

Profesor y tutores, para poder ingresar desde sus navegadores, por favor contactarnos para facilitarles la IP. El puerto para ingresar al dashboard es el 8550.

La forma final de la dirección seria: http://IP_PUBLICA:8550


-------------------------------------------------------------

Para instalar y ejecutar el dashboard localmente, por favor leer el archivo: Manual de ejecución  local- Dashboard, ubicado en la carpeta "otros". De igual forma, este manual de instalación se enviara en el .zip de la entrega final del proyecto.
